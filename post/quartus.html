Instalando o Quartus II no Ubuntu 19.10

Olá! Sei que instalar o Quartus II envolve algum trabalho, e dá sempre algum erro, deste modo vim compartilhar com você as etapas que precisei para fazer essa instalação, então vamos lá!

  1. Passo: O Quartus precisa de dependências 32 bits, assim o sistema precisa ser preparado assim execute os seguintes comando no terminal: sudo dpkg --add-architecture i386 sudo apt update sudo apt install libxft2:i386 libxext6:i386 libncurses5:i386 bzip2:i386
  2. Passo: libpng não está sendo atualizada a partir do Ubuntu 16.04 então utilizar sudo apt install, não é uma opção. Tente: sudo wget http://ftp.us.debian.org/debian/pool/main/libp/libpng/libpng12-0_1.2.50-2+deb8u3_amd64.deb $ sudo dpkg -i libpng12-0_1.2.50-2+deb8u3_amd64.deb Obs: Caso isso não funcionar continue este tutorial, mas para frente damos outra solução.
  3. Passo: Agora faremos o download do Quartus no site da Intel, precisa ser a opção Lite, pois a mesma é gratuita. https://fpgasoftware.intel.com/13.1/?edition=web&platform=linux
  4. Passo: Quando o download estiver finalizado descompacte o arquivo, após no terminal entre na pasta que foi descompactada:

    cd altera

    cd Components

    ./QuartusSetupWeb-13.1.0.162

    A tela de instalação será aberta, faça download apenas dos devices da placa que você utilizará. Além disso deixe a opção Prime Help e ModelSim deselecionado.
  5. Passo: para instalar o Quartus Help execute no terminal: ./QuartusHelpSetup-13.1.0.162
  6. Passo: Agora instalando o ModelSim: ./ModelSimSetup-13.1.0.162
  7. Passo: Agora vamos executar cd altera/13.1/quartus/bin ./quartus Agora se der erro por causa da libpng12, execute no terminal: export QUARTUS_64BIT=1
  8. Passo: USB Blaster: esse passo é extremamente importante sudo 'gedit /etc/udev/rules.d/altera-usb-blaster.rules Quando o arquivo abrir copie as linhas a seguir e cole no arquivo: ATTR{idVendor}=="09fb", ATTR{idProduct}=="6001", MODE="666" ATTR{idVendor}=="09fb", ATTR{idProduct}=="6002", MODE="666" ATTR{idVendor}=="09fb", ATTR{idProduct}=="6003", MODE="666" ATTR{idVendor}=="09fb", ATTR{idProduct}=="6010", MODE="666" ATTR{idVendor}=="09fb", ATTR{idProduct}=="6810", MODE="666” Após você precisa testar se o USB está funcionando, execute: cd /altera/13.1/quartus/bin ./jtagconfig O retorno deve ser similar a “1) USB-Blaster [3-3]” Obs: Existe a opção de criar um ícone para não precisar navegar até as pastas, mas não é necessário
